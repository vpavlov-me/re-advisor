## Quick Reference

### Key Concepts

| Concept | Description | Example |
|---------|-------------|---------|
| **Channel** | Communication pathway for real-time updates | `workshop:123:messages` |
| **Presence** | Track online/offline status | `{ participant_id: 1, online: true }` |
| **Broadcast** | Send events to all subscribers | `broadcastTyping(channel, ...)` |
| **CDC** | Change Data Capture - database triggers | Postgres â†’ Realtime |
| **Subscription** | Listen for real-time events | `subscribeToMessages(...)` |

### Channel Types

| Channel | Purpose | Data Type |
|---------|---------|-----------|
| `workshop:X:presence` | Online/offline status | Presence events |
| `workshop:X:messages` | Chat messages | Postgres CDC |
| `workshop:X:typing` | Typing indicators | Broadcast |
| `workshop:X:cursors` | Cursor positions | Broadcast |
| `workshop:X:values` | Value selections | Postgres CDC |
| `workshop:X:votes` | Voting updates | Postgres CDC |
