## Question Design Principles

### 1. Zero Defaults
- All questions start with no selection
- Forces intentional choice
- Improves data quality

### 2. Consensus-Sensitive Questions
Marked questions are specifically designed to reveal divergence:
- Q2.5: Financial information adequacy
- Q4.1: Decision-making clarity
- Q7.4: Understanding of inheritance

### 3. Conditional Logic
Some questions appear only if:
- Q6.2 only if Q6.1 shows existing structures
- Q3.X questions adapt based on generation of respondent

### 4. Inverted Scoring
Questions where higher number = worse outcome:
- Q4.4: Frequency of unresolved conflicts
- Automatically inverted in scoring algorithm

### 5. Weighting
- **2.0 = Critical** - Most important questions
- **1.5 = High** - Very important
- **1.0 = Standard** - Normal weight
- **0.5 = Low** - Supporting context
